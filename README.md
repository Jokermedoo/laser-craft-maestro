
# 🏢 موقع ورشة المعز لخدمات الليزر

## 📋 نظرة عامة على المشروع

موقع ويب متقدم ومتجاوب تماماً لورشة المعز المتخصصة في خدمات الليزر في صعيد مصر. يتميز الموقع بتصميم عصري وواجهة مستخدم متطورة مع لوحة تحكم إدارية شاملة.

## ✨ المميزات الرئيسية

### 🎨 التصميم والواجهة
- **تصميم متجاوب بالكامل** - يعمل بشكل مثالي على جميع الأجهزة
- **حركات متقدمة** - انتقالات سلسة وتأثيرات بصرية جذابة
- **نظام ثيمات متقدم** - إمكانية تخصيص الألوان والمظهر
- **واجهة عربية** - دعم كامل للغة العربية مع RTL

### 🛠️ الوظائف الأساسية
- **معرض الأعمال** - عرض تفاعلي لأحدث المشاريع مع فلترة متقدمة
- **صفحة الخدمات** - عرض تفصيلي لجميع خدمات الليزر
- **حاسبة الأسعار** - أداة تفاعلية لحساب تكلفة المشاريع
- **نظام العروض** - عروض يومية وخاصة مع مؤقت تنازلي
- **الأسئلة الشائعة** - قسم تفاعلي للإجابة على استفسارات العملاء

### 🎯 المميزات التسويقية
- **أزرار WhatsApp مباشرة** - تواصل سريع مع العملاء
- **عروض ترويجية** - بانرات جذابة للعروض الخاصة
- **إحصائيات مميزة** - عرض أرقام الإنجازات والعملاء
- **شهادات العملاء** - مراجعات وتقييمات العملاء

### ⚙️ لوحة التحكم الإدارية
- **محرر الثيمات المتقدم** - تحكم كامل في الألوان والتصميم
- **محرر المحتوى** - إدارة سهلة لجميع نصوص الموقع
- **إدارة المعرض** - رفع وتنظيم صور الأعمال
- **إدارة الخدمات** - تحديث أوصاف وأسعار الخدمات
- **مراقب الأداء** - تحليل أداء الموقع في الوقت الفعلي
- **نظام الإشعارات** - تنبيهات ذكية للمدير
- **الأمان المتقدم** - حماية متعددة المستويات

## 🚀 التقنيات المستخدمة

### Frontend Framework
- **React 18** - مكتبة واجهة المستخدم الحديثة
- **TypeScript** - لضمان جودة الكود وقابليته للصيانة
- **Vite** - أداة البناء السريعة والحديثة

### UI/UX
- **Tailwind CSS** - إطار عمل CSS المرن والسريع
- **Shadcn/UI** - مكونات واجهة مستخدم جاهزة وقابلة للتخصيص
- **Framer Motion** - مكتبة الحركات والانتقالات المتقدمة
- **Lucide React** - مجموعة شاملة من الأيقونات

### Data Management
- **React Query** - إدارة البيانات وحالة الخادم
- **Context API** - إدارة الحالة العامة للتطبيق
- **Local Storage** - تخزين محلي للإعدادات والبيانات

### Features & Tools
- **DND Kit** - نظام السحب والإفلات
- **React Hook Form** - إدارة النماذج والتحقق من صحة البيانات
- **Sonner** - نظام الإشعارات المتقدم
- **Date-fns** - معالجة التواريخ والأوقات

```
src/

│   ├── admin/          # مكونات لوحة التحكم

│   ├── services/       # مكونات الخدمات

│   ├── gallery.ts     # بيانات المعرض
│   └── services.ts    # بيانات الخدمات

├── contexts/          # React Context للحالة العامة
├── pages/             # صفحات التطبيق
├── types/             # تعريفات TypeScript
└── utils/             # أدوات مساعدة
```

## 🛠️ التثبيت والتشغيل

### متطلبات النظام
- Node.js (الإصدار 18 أو أحدث)
- npm أو yarn أو bun

### خطوات التثبيت

1. **استنساخ المستودع**
```bash
git clone [repository-url]
cd laser-workshop-website
```

2. **تثبيت التبعيات**
```bash
npm install
# أو
yarn install
# أو
bun install
```

3. **تشغيل الخادم المحلي**
```bash
npm run dev
# أو
yarn dev
# أو
bun dev
```

4. **فتح المتصفح**
اذهب إلى `http://localhost:5173`

## 🔧 التخصيص والتطوير

### إضافة خدمات جديدة
1. افتح ملف `src/data/services.ts`
2. أضف الخدمة الجديدة إلى مصفوفة `servicesData`
3. تأكد من إضافة جميع الخصائص المطلوبة

### إضافة أعمال جديدة للمعرض
1. افتح ملف `src/data/gallery.ts`
2. أضف العمل الجديد إلى مصفوفة `galleryData`
3. رفع الصورة إلى مجلد `public/lovable-uploads/`

### تخصيص الألوان والثيمات
1. استخدم مكون `ThemeSwitcher` لتغيير الألوان
2. أو عدل ملف `src/constants/defaultTheme.ts`
3. أو استخدم لوحة التحكم الإدارية

### إضافة صفحات جديدة
1. أنشئ مكون جديد في `src/pages/`
2. أضف المسار الجديد في `src/App.tsx`
3. أضف رابط للصفحة في مكون `Navbar`
